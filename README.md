## Testing the server locally
#### Go go the Config folder and change the development.yml file and add your local database information

### Example
```
db:
  host: 'localhost'
  username: 'postgres'
  password: 'postgres'
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
